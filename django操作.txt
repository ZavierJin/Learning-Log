
	激活虚拟环境
	先进入Learning-log目录，再ll_env\Scripts\activate

	修改model.py，在里面添加类

	迁移数据库
	python manage.py makemigrations learning_logs
	
	应用迁移
	python manage.py migrate
	
	在admin.py中注册模型
	
	运行
	python manage.py runserver
	退出	ctrl + c
	
	启动django shell
	python manage.py shell
	退出	输入ctrl + z 再按回车
